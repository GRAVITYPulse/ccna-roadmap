# CCNA v7 – Enterprise Networking, Security, and Automation (ENSA)

## 📖 Overview
This repository contains my structured study roadmap, Packet Tracer labs, troubleshooting exercises, and notes for Cisco CCNA v7 – Enterprise Networking, Security, and Automation (ENSA).

ENSA focuses on:
- OSPF routing
- ACLs
- NAT
- WAN technologies
- VPN concepts
- QoS
- Network management
- Enterprise troubleshooting

---

# 🎯 Objectives

By completing this section, I should be able to:

- Configure and troubleshoot OSPF
- Configure ACLs
- Configure NAT and PAT
- Understand WAN technologies
- Understand VPN concepts
- Understand QoS concepts
- Manage enterprise devices
- Troubleshoot enterprise networks
