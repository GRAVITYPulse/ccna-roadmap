# CCNA v7 – Switching, Routing, and Wireless Essentials (SRWE)

## 📖 Overview
This repository contains my structured study roadmap, Packet Tracer labs, troubleshooting exercises, and notes for Cisco CCNA v7 – Switching, Routing, and Wireless Essentials (SRWE).

SRWE focuses on:
- VLANs
- Inter-VLAN routing
- STP
- EtherChannel
- DHCP
- WLANs
- Switch security
- Static routing

---

# 🎯 Objectives

By completing this section, I should be able to:

- Configure VLANs and trunks
- Configure inter-VLAN routing
- Understand STP operation
- Configure EtherChannel
- Configure DHCPv4
- Understand wireless LAN concepts
- Secure switches
- Configure static routes
- Troubleshoot Layer 2 and Layer 3 issues
