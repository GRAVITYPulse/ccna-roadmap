# DNS

## What is DNS?
Converts domain names → IP addresses

---

## Example
google.com → 142.250.x.x

---

## Test
ping google.com

* DNS resolves:

  * google.com → IP

---

## 🧠 Explanation

Instead of remembering IPs, we use names.
