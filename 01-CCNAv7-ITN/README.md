# CCNA v7 – Introduction to Networks (ITN)

## 📖 Overview
This repository contains my structured study roadmap, Packet Tracer labs, troubleshooting exercises, and notes for Cisco CCNA v7 – Introduction to Networks (ITN).

ITN focuses on:
- Networking fundamentals
- Basic device configuration
- IPv4 and IPv6 addressing
- Ethernet switching
- Network protocols
- Basic troubleshooting

---

# 🎯 Objectives

By completing this section, I should be able to:

- Understand network architectures
- Configure Cisco IOS devices
- Understand OSI and TCP/IP models
- Configure switches and routers
- Understand IPv4 and IPv6 addressing
- Perform subnetting
- Verify network connectivity
- Troubleshoot basic network issues
