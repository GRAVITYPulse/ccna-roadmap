# IPv4 Addressing

* IP structure
* Network vs Host

---

## 🧠 Explanation

Example:

```
192.168.1.10/24
```

* Network: 192.168.1.0
* Host: 10

👉 Subnet mask defines network boundary

## Key Terms
- IP = device address
- Subnet mask = defines network boundary
- Gateway = exit point

---

## Rule
Devices must be in same network to communicate without router
