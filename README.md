# ccna-study-journal

# CCNA v7 Networking Study Roadmap (ITN → SRWE → ENSA)

## 📘 Overview

This repository is a structured, hands-on learning journal covering the full **Cisco CCNA v7 curriculum**:

- ITN (Introduction to Networks)
- SRWE (Switching, Routing & Wireless Essentials)
- ENSA (Enterprise Networking, Security & Automation)

It is designed to go beyond theory and focus on:

- Real packet behavior analysis
- Cisco IOS CLI mastery
- Network troubleshooting skills
- Subnetting and IP design
- Layer 2 and Layer 3 understanding
- Hands-on Packet Tracer simulations

---

## 🎯 Purpose

The goal of this repository is to build:

- 🧠 Strong networking fundamentals
- ⚙️ Practical Cisco CLI skills
- 🔍 Troubleshooting mindset
- 🌐 Real-world network understanding
- 💼 Job-ready junior network engineer skills
- 🏆 CCNA certification readiness

---

## 🧠 Learning Philosophy

This is NOT a memorization-based repo.

Instead, it follows:

- 📖 Learn concepts
- 🧪 Build Packet Tracer labs
- 🔍 Break and troubleshoot networks
- 🧩 Analyze packet flow behavior
- 📝 Document everything in GitHub
- 🔁 Repeat until mastery
